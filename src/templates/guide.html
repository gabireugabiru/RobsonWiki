Getting started

To start of create a main.robson, then write the following

robson robson robson
comeu 65
robson robson robson robson robson robson robson

The first line has 3 robsons, so it is the push command.
The push command has 1 param, number params are normally written with comeu {number}.
The push command adds the value of param1 to the stack.

At the third line we have 7 robsons, so it is the print command
The print command has no params
The print command removes the top of the stack and prints it as ascii

The result of this code is a "A" with no newline since 65 is equal to A
